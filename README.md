# Random Password Generator

A sleek, secure, and customizable random password generator built to help users create strong, uncrackable passwords. This tool allows users to define password length and toggle various character sets (uppercase, lowercase, numbers, and special characters) to meet specific security requirements.

## 🚀 Features

* **Customizable Length:** Generate passwords of any desired length.
* **Character Selection:** Toggle inclusion of uppercase letters, lowercase letters, numbers, and special symbols (`@`, `#`, `$`, etc.).
* **Strength Indicator:** Instantly evaluates how secure the generated password is.
* **One-Click Copy:** Copy the generated password straight to your clipboard.
* **Locally Secure:** No data is sent to any server; passwords are generated entirely on your local machine.
